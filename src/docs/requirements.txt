1. Загальна концепція

Тип застосунку: SPA (Single Page Application)
Технологія: Angular (остання LTS)
Призначення:
Інтерактивна вебплатформа на тему гастротуризму равликових ферм із вбудованим бізнес-калькулятором.

Контент базується на курсовій роботі:

гастротуризм
ферми (наприклад, Ravlyk Park)
маршрути
страви
бізнес-проєкт

2. Архітектура застосунку
2.1 Структура модулів

Розбити на lazy-loaded модулі:

/core
/shared
/features
  /home
  /about-tourism
  /farms
  /farm-details
  /route
  /calculator
  /gallery

2.2 Маршрутизація

Обов'язкові маршрути:

/                → Головна
/tourism         → Про гастротуризм
/farms           → Список ферм
/farms/:id       → Деталі ферми
/route           → Маршрут
/calculator      → Бізнес-калькулятор
/gallery         → Страви / дегустації
